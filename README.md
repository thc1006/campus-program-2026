# Campus Integration & Drone Innovation

## 專案概述 (Overview)

本專案目前分為兩大核心發展方向：

1. **校園生活整合系統 (Web/App)**：以學生為中心，整合校內各行政單位的資訊與服務（如校車、飲膳）。
2. **無人機創新研發 (Drone R&D)**：包含飛手考照計畫與遙控器硬體市場開發。

同時，團隊將以此專案為基礎，籌組分隊參與商業競賽（如 U-start）。

## 行政對接與合作單位 (Stakeholders)

目前正積極洽談或預計接觸的校內單位：

* **服學中心 (Service-Learning Center)**：可能涉及服務學習時數或相關合作。
* **生輔組 (Student Assistance Division)**：針對開學典禮的合作機會。
* **博雅中心 (Center for General Education)**：經費申請與課程相關資源。
* **飲膳組 (Dining Division) & 事務組**：整合「飲膳資訊」與「校車時刻表」至學生系統中。
